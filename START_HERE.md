# 🎮 START HERE - CNZR Framework Website

## 👋 Welcome!

Selamat datang di CNZR Framework Website! Ini adalah website profile dengan design pixel art retro yang menampilkan semua fitur CNZR framework.

## ⚡ Quick Start (3 Steps)

### 1️⃣ Install Dependencies
```bash
npm install
```

### 2️⃣ Build CSS
```bash
npm run build
```

### 3️⃣ Start Server
```bash
npm run dev
```

### 4️⃣ Open Browser
```
http://localhost:3000
```

**That's it! 🎉**

## 🐛 Troubleshooting

### Server Won't Start
```bash
# Check Node version
node --version  # Should be 16+

# Reinstall dependencies
rm -rf node_modules
npm install
```

### CSS Not Loading
```bash
# Rebuild Tailwind
npm run build

# Check output file
ls -la public/output.css
```

### Port Already in Use
```bash
# Change port
PORT=3001 npm run dev
```
